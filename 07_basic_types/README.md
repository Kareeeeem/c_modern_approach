# Basic types notes

## Int

By default `int` types are signed.

16 bit machine

Type | min | max
---- | --- | ---
`short int` | -32,768 | -32,768
`unsigned short int` | 0 | 65,535
`int` | -32,768 | 32,768
`unsigned int` | 0 | 65,535
`long int` | -2,147,483,648 | 2,147,483,648
`unsigned long int` | 0 | 4,294,967,295
